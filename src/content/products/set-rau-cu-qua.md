## Mô tả chi tiết

Bộ rau củ quả mini giúp bé làm quen với các loại thực phẩm qua hình thức chơi mà học.

### Thông tin sản phẩm

- **Số lượng:** 5-8 món tùy set
- **Kích thước:** 5-8cm mỗi món
- **Chất liệu:** Len cotton, bông gòn

### Các món trong set tiêu chuẩn

1. Cà rốt
2. Bắp cải
3. Cà chua
4. Ớt chuông
5. Bông cải xanh

Có thể đặt thêm các món khác theo yêu cầu!
